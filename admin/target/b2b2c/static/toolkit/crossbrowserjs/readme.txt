1.excanvas.js主要解决IE9以下的版本是不支持html5的，及canvas绘图。
2.html5shiv.js主要解决HTML5提出的新的元素不被IE6-8识别，这些新元素不能作为父节点包裹子元素，并且不能应用CSS样式。
3.Respond.js让不支持css3 Media Query的浏览器包括IE6-IE8等其他浏览器支持查询。

使用方式：
===============================================================================
<!--[if lt IE 9]>
	<script src=".../html5shiv.js"></script>
	....
<![endif]-->
===============================================================================
